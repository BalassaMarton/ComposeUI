<Hero title="ComposeUI">
ComposeUI is a .NET based general UI Container and Unified UI and App host which
enables the hosting of Web and desktop content. It supports desktop and web applications
in order to provide an evergreen alternative to Electron, OpenFin and similar by the use of WebView2.
